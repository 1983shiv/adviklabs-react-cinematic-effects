# ✨ react-cinematic-effects

**Drop-in, extensible cinematic UI effects for React.**

This Storybook is the interactive playground and living documentation for every
effect in the library. Use it to:

- **Explore** each effect visually before adding it to your project
- **Tweak** every prop in real-time via the **Controls** panel
- **Check** accessibility in the **A11y** tab
- **Read** generated API docs in the **Docs** view

---

## Available Effects

- **AccordionSlider** | Horizontal / vertical image panels that expand on hover or click
- **ImageTrail**      | Cursor-following image tiles that spawn and fade as you move
- **StickyStack**     | Scroll-driven feature showcase with a sticky paired visual
- **FlipCards**       | Responsive 3D cards that flip to reveal more info on hover or click
- **CircularText**    | Text rendered along a rotating SVG arc — spin, scroll-reactive, or static
- **ColorShift**      | Scroll-driven background and text colour transitions between page sections
- **TypeWriter**      | Phrase cycler, viewport-triggered code block, and staggered chat bubble sequence

## Quick Start

```bash
npm install react-cinematic-effects
```

```tsx
import { AccordionSlider } from 'react-cinematic-effects';
// optional: import global CSS tokens
import 'react-cinematic-effects/styles.css';

export default function Hero() {
  return (
    <AccordionSlider
      items={[
        { id: '1', backgroundSrc: '/img/a.jpg', title: 'Strategy' },
        { id: '2', backgroundSrc: '/img/b.jpg', title: 'Design' },
        { id: '3', backgroundSrc: '/img/c.jpg', title: 'Deliver' },
      ]}
    />
  );
}
```

```tsx
import { CircularText } from 'react-cinematic-effects';

export default function Badge() {
  return (
    <CircularText text="DESIGN • BUILD • SHIP • SCALE • " repeat={2} size={280}>
      <span>2026</span>
    </CircularText>
  );
}
```

```tsx
import { TypeWriter } from 'react-cinematic-effects';

export default function Hero() {
  return (
    <TypeWriter
      variant="cycler"
      phrases={['We craft digital experiences.', 'We define modern brands.', 'We architect future products.']}
      typeSpeed={60}
      showCursor
      cursorColor="#c8a97e"
    />
  );
}
```

```tsx
import { ColorShift } from 'react-cinematic-effects';

export default function Page() {
  return (
    <ColorShift
      sections={[
        { bg: '#0a0a0b', text: '#eae7e2', children: <HeroSection /> },
        { bg: '#081a12', text: '#dff0e8', children: <ForestSection /> },
        { bg: '#0d081a', text: '#e0dff0', children: <NightSection /> },
      ]}
    />
  );
}
```

## Philosophy

- **Zero mandatory dependencies** — no GSAP, no Framer Motion, no heavy runtime
- **SSR-safe** — every component renders a valid HTML fallback on the server
- **Tree-shakeable** — import only the effect you ship
- **Fully accessible** — ARIA roles, keyboard support, and `prefers-reduced-motion` built in
- **CSS-customisable** — every visual variable is a CSS custom property
